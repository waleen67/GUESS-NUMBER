# 🎯 Guess the Number Game

A simple Python game where the player tries to guess a randomly generated number within a given range.

## 📌 Features

* Random number generation
* Input validation using `isdigit()`
* Feedback for each guess (too high / too low)
* Counts number of attempts
* Range checking

## 🎮 How to Play

* Enter a number between 1 and 100
* The program will tell you if your guess is:

  * Too high 📈
  * Too low 📉
  * Correct ✅
* Keep guessing until you find the correct number

## 🧠 Concepts Used

* Loops (`while`)
* Conditionals (`if/elif/else`)
* Functions like `isdigit()`
* Random number generation
